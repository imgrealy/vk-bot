# Простейший бот для ответа на сообщения вконтакте, работает через Callback API.
Для настройки необходимы следующие шаги:
1) Создаем ключ сообщества в Управление - Работа с API, достаточно доступа к сообщениями сообщества.
    Данный ключ вставляем в api.token в application.properties
2) Запускаем туннель для локальной машины (я использовал ngrok) по порту, который указан в server.port в application.properties
    Для получившегося туннеля добавляем Callback API (там же в "Работа с API"), к адресу добавляем /bot, например https://bd1910d60677.ngrok.io/bot
    Версию указываем 5.120. 
    В application.properties приложения вставляем api.confirmation, который указан вконтакте как "Строка, которую должен вернуть сервер: {код}"
    В типы событий добавляем "Входящие сообщения"
3) Запускаем приложение и только после этого нажимаем подтвердить в Callback API.
   Если все успешно, то адрес будет подтвержден, после этого необходимо включить сообщения сообщества, бот начнет на них отвечать.
   
Версия прописана жестко, так как не реализована совместимость.
Классы, которые описывают объекты, приходящие от Callback API, реализованы только для функционала подтверждения сервера и ответа на новое сообщение, так как api вконтакте обширно и реализацию всех возможный вариаций не счел необходимым.
